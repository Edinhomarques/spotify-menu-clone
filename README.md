# Spotify-menu-clone
 This project presents a copy of Spotify's home screen, the purpose of this project was to understand more about front-end, notions of desing and web technologies. 
 
 Tecnologies used: 
 * HTML5
 * CSS3
 * Javascript
 * Bootstrap

## How to run
### Go to Spotify-menu-clone directory:

>cd spotify-menu-clone

### With python 3 run:
> py -m http.server 3000 or simply open the index.html file in your browser
